# ACDC-Cardiac-Segmentation: Based On Unet And variations

## 如何运行本项目？

本项目保守的采用Conda环境进行环境容器配置，为了快速开始，请您按照本项目提供的environment.yml文件进行环境复刻。由于本项目的依赖非常庞大，请您耐心等待下载

```bash
# 按照本项目导出的environment.yml进行环境配置
# 项目使用的是固定的Python 3.10
conda env create -f environment.yml -n acdc_cardiac
conda activate acdc_cardiac
```
